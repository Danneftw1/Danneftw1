# Daniel Nilsson :computer::robot::musical_note:

![Banner](banner.png)

I'm a developer with a background in AI and machine learning who has grown into a full-stack engineer across the modern web and cloud stack. These days most of my work sits at the intersection of **LLM engineering** and **product development** — designing AI pipelines, building the services around them, and shipping them to real users.

Currently working as an innovation developer at the consulting firm **Quokka** in Gothenburg, Sweden, where I build AI-powered products end-to-end — from prompt design and multi-stage LLM pipelines, to the TypeScript/Python services that run them, to the Azure infrastructure underneath.

I'm also quite the drummer, and I love music :wink::chopsticks:

## :sparkles: What I'm working on

- **AI-powered SaaS platforms** — designing and shipping production LLM features on Azure (OpenAI / AI Foundry)
- **Multi-stage AI pipelines** — prompt engineering, parallelized model orchestration, retries & content-filter handling
- **Full-stack TypeScript** — React + Vite frontends, Express services, and Python/Flask AI services in a pnpm monorepo
- **Cloud infra as code** — Azure Cosmos DB, Key Vault, B2C / External ID, App Service, Bicep, GitHub Actions
- **AI developer tooling** — MCP servers, Claude Code / Cursor workflows, AI-assisted code review and PR automation
- **LLM observability** — tracing, evals, and prompt iteration with Langfuse
